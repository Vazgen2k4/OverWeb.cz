Обо мне

Арутюнян Вазген Арамович.

Свой путь в IT я начал в 2017м году с изучения такого языка программирования как: C++ по советом "экспертов". После чего посвятил свою деятельность изучению Web-разработке. За это время мой стек знаний пополнился такими технологиями как: Html5, Css3, JavaScript + jQuery, Bootstrap 4/5, Node js, git, php и Vue. Помимо этого последний год занимаюсь разработкой кроссплатформенных приложений на технологии Flutter с использованием языка программирования Dart.
